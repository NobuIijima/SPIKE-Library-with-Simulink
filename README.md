## Installation
The following four software programmes need to be installed.
  1. MATLAB
  2. Cygwin
  3. Arm GNU Toolchain
  4. USB driver (Zadig)

Set up the environment with reference to the ‘Installation instructions.pdf’ in the ‘docs’ folder.
